# 🚀 AI Resume Analyzer

An AI-powered Resume Analyzer that evaluates resumes based on ATS scoring, extracts key skills, and provides job matching insights.

---

## 🧠 Features

- 📄 Upload Resume (PDF)
- 📊 ATS Score Calculation
- 🧾 Skill Extraction
- 💼 Job Role Matching
- 🤖 AI-based Feedback

---

## 🛠️ Tech Stack

- **Frontend:** React.js  
- **Backend:** Flask (Python)  
- **Libraries:** PyPDF2, Flask-CORS  
- **Tools:** Git, GitHub
